# VC Rechner

## GitHub Pages
1. Neues GitHub-Repository erstellen
2. index.html hochladen
3. Settings → Pages
4. Deploy from Branch → main
5. Speichern

Danach ist der Rechner unter einer festen URL erreichbar.

## Replit
1. Neues HTML/CSS/JS Projekt erstellen
2. index.html ersetzen
3. Run klicken

Änderungen werden automatisch gespeichert.
